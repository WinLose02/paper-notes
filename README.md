# Paper-notes
This repository contains my paper review notes on machine learning. (Primarily LLM, RL, Generative Models)

I categorize my notes into three types:
* One is for papers or documents which I read carefully all the way to the end. In this case, I'll organize a note on Notion(public accesible). I'll provide the link in the notes.
* Another is for those which I read briefly on Abstract, Introduction, and Conclusion. 
* The other is for those which I made a mini project for them. I'll provide the github repository link in the notes.
